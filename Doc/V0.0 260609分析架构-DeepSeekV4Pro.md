## 三、架构优点

1. **数据中心模式** — `sensor_data_mgr` 作为共享数据中心，耦合度低，各 Task 通过统一接口读写
2. **BSP 抽象层** — 新增的 `bsp.c/h` 将 LED 硬件操作从业务逻辑中解耦，符合分层架构原则
3. **状态机模式** — 主业务使用显式状态机，Doc 中有完整设计文档，为复杂业务扩展预留空间
4. **FreeRTOS 原生特性** — Queue（命令传递）、EventGroup（WiFi状态通知）、Mutex（数据保护）使用得当
5. **模块化分工** — WiFi/MQTT/传感器/业务四个模块独立编译和任务运行
